# Secretos detrás de rest en 2023

## Ejemplo práctico

Aquí hay un ejemplo de código que muestra cómo puedes usar rest:

```ruby
def show
  @widget = Widget(params[:id])
  respond_to do |format|
    format.html # show.html.erb
    format.json { render json: @widget }
  end
end
```
