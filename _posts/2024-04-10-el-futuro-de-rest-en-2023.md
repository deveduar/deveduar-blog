## ¿Qué es Laravel?

Laravel es una herramienta/paquete/framework que ha revolucionado la forma en que trabajamos en desarrollo moderno.

## Ejemplo práctico

Aquí hay un ejemplo de código que muestra cómo puedes usar laravel:

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print(fibonacci(10))
```
