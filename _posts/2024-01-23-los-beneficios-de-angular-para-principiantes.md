## Ejemplo práctico

Aquí hay un ejemplo de código que muestra cómo puedes usar jekyll:

```ruby
def show
  @widget = Widget(params[:id])
  respond_to do |format|
    format.html # show.html.erb
    format.json { render json: @widget }
  end
end
```
