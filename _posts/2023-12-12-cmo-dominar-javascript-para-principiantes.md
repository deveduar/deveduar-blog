## Conclusión

Dominar fullstack puede abrir muchas oportunidades en tu carrera como desarrollador. ¡No dudes en probarlo!

## Ejemplo práctico

Aquí hay un ejemplo de código que muestra cómo puedes usar fullstack:

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print(fibonacci(10))
```
