# Secretos de ci en producción

## Introducción

Con yarn, puedes resolver problemas complejos y crear soluciones escalables.

## Código de ejemplo

```python
def factorial(n): return 1 if n == 0 else n * factorial(n-1)
print(factorial(5))
```
