## Ejemplo práctico

Aquí hay un ejemplo de código que muestra cómo puedes usar rest:

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print(fibonacci(10))
```
